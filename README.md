## Arduino Controller 8266

This project is a web-based controller for an ESP8266 microcontroller. It allows you to control an LED connected to the ESP8266, including toggling the power and adjusting the brightness.

### Features

- Toggle LED power on/off
- Adjust LED brightness
- Check connection status with the ESP8266

### Getting Started
